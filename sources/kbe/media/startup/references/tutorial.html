Инструкция использования редактора SC-g конструкций

Шаг 1. Создание узлов

Выбираем пункт «режим выделения». Двойным кликом левой кнопкой мыши создаём константный sc.g-узел.

Шаг 2. Изменение типа узла

Правой кнопкой мыши кликаем по узлу, появляется окно выбора действия. В пункте «Изменить тип» выбираем нужный тип узла.

Шаг 3. Установка идентификатора узла

Нажав клавишу «I» или выбрав пункт «изменить идентификатор» после нажатия правой кнопки мыши по узлу, устанавливается идентификатор узла.

Шаг 4. Создание ещё одного узла

Создаём еще один sc.g-узел, как в пункте 1. Устанавливаем тип «множества», как в пункте 2, и указываем идентификатор, как в пункте 3.

Шаг 5. Создание дуги

Выбираем инструмент «режим создания дуги». Кликаем по объекту, из которого должна выходить дуга, затем кликаем по объекту, куда дуга должна входить.

Шаг 6. Изменение типа дуги

Правой кнопкой мыши кликаем по дуге, появляется окно выбора действия. В пункте «Изменить тип» выбираем нужный тип дуги.

Результат :

Шаг 7. Установка идентификатора дуги

Нажав клавишу «I» или выбрав пункт «изменить идентификатор» после нажатия правой кнопки мыши по дуге, устанавливается идентификатор дуги.

Шаг 8. Создание дуги, входящей в другую дугу

Следуя пунктам 1-3, создаем узел с типом «отношение» и идентификатором «включение*». Как в пункте 5 создаем sc.g-дугу принадлежности, которая исходит из узла и входит в другую дугу.

Шаг 9. Установка содержимого узла

Создаём еще один sc.g-узел и устанавливаем идентификатор. По аналогии с 5 пунктом создаём дугу принадлежности. Установить содержимое можно нажав клавишу «С» или выбрав пункт «установить содержимое», после нажатия правой кнопкой мыши по узлу. Содержимое может быть 3 типов: изображение, строковая переменная, число.

Шаг 10. Просмотр содержимого

Просмотр содержимого осуществляется при нажатии клавиши «H» или выбрав пункт «просмотреть содержимое», после нажатия правой кнопкой мыши по узлу.

Шаг 11. Создание контура

Выбираем инструмент «Режим создания sc.g-контура». Создаём контур произвольной формы, выбирая любые точки в поле редактирования. Заканчиваем создание контура кликом левой клавиши мыши на начальной точке. Соединяем дугой вершину и контур.

Шаг 13. Создание конструкции внутри контура

Следуя шагам 1-7, создаём произвольную конструкцию внутри контура.

Шаг 14. Выравнивание по сетке

Выбираем инструмент «Выравнивание по сетке». Настраиваем выравнивание так, как нам нужно, жмём кнопу «ОК».
